## 3. 关键政策与信号（按国家）

### 3.1 越南

- 信号：外资项目更强调可解释性与文件留存，审批周期拉长的概率上升。
- 对业务的影响：
  - 新项目落地：审批时间与合规成本上升
  - 供应链：高敏领域需要更清晰的原产地与客户去向解释
- 建议动作：
  - 建立“审批材料清单 + 证据链模板”
  - 将审批周期写入财务模型

### 3.2 柬埔寨

- 信号：同一政策在地区与部门层面的执行差异更值得关注。
- 对业务的影响：
  - 单点风险上升：依赖单一地区/单一部门路径的业务更脆弱
- 建议动作：
  - 做业务路径分散与备选方案
  - 建立本地信息网络与事件响应机制

## 4. 行业专项（示例：制造业）

- 风险：外资审查、出口管制、原产地规则、供应链透明度要求上升
- 机会：产业链重组带来的配套需求增加
- 建议动作：
  - 强化供应链数据化与可追溯
  - 与本地园区/供应商建立长期合作，提升本地化率

## 5. 下周期观察名单（未来 7-30 天）

- 外资审批与税务稽查的执行强度是否升级
- 区域资金流向与合规规则变化
- 重点项目是否出现制裁或舆论风险外溢

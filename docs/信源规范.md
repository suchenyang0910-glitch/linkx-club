# linkx.club 信源规范（v1）

## 1. 目标

- 保证内容可追溯、可复核、可解释
- 降低“单一来源误导”风险
- 让读者清楚区分事实、推断与观点

## 2. 信源分层

建议把信源分为 4 层：

- A 层：官方发布（政府公告、央行/监管机构、法律文本、统计局）
- B 层：权威媒体与区域媒体（国际主流媒体、区域财经媒体、当地大型媒体）
- C 层：行业报告与机构研究（咨询公司、协会、券商、智库公开报告）
- D 层：社区与二手信息（社交媒体、论坛、群聊、匿名线索）

使用原则：

- A/B 层优先
- C 层用于补充框架、数据与行业判断
- D 层只能作为线索，不可单独作为结论依据

## 3. 多源交叉规则

- 重要事实：至少 2 个独立来源交叉
- 涉及敏感结论：至少包含 1 个 A 层或 B 层来源
- 争议信息：必须注明“存在不同说法”，并列出各方口径差异

## 4. 引用与标注

建议文末统一增加：

- 数据截止日期
- 引用来源清单（按 A/B/C/D 分组）
- 关键变量与不确定性说明

## 5. 写作分层（必须）

每篇文章至少包含以下三段之一：

- 事实：发生了什么（可引用）
- 推断：为什么会这样（逻辑链）
- 观点：我们怎么判断（明确是观点）

## 6. 匿名线索（留言/爆料）处理

- 默认不公开展示
- 先进入审核队列
- 满足“可公开”条件后才可摘要公开（仍需去标识化）
- 永远不展示完整 IP 或可识别个人信息

## 7. 风险等级与审核

建议按内容风险做分级：

- 低：生活类、公开信息汇总
- 中：政策解读、行业影响
- 高：制裁、军事、敏感人物、未证实线索

中/高风险建议增加人工复核与措辞审慎检查。

# linkx.club 每日简报模板（v1）

## 标题

【每日简报】YYYY-MM-DD：一句话概括主线（国家/主题）

## 0. 一句话判断（必须）

用一句话告诉读者：今天最重要的信号是什么，它对谁有影响。

## 1. 今日重点（3-5 条）

每条都按固定结构输出：

- 事件：发生了什么
- 信源：A/B/C/D（可列 1-3 个）
- 影响：对谁有影响（行业/企业/投资/合规）
- 判断：我们倾向怎么解读（清晰标注为观点）
- 观察点：接下来 1-3 天要看什么

## 2. 国家/行业快速雷达（可选）

- 越南：一句话
- 泰国：一句话
- 新加坡：一句话
- 马来西亚：一句话
- 柬埔寨：一句话
- 老挝：一句话

## 3. 今日延伸阅读（可选）

- 站内深度分析 1 篇
- 站内国家观察页 1 个
- 外部参考 1-2 个（注明信源层级）

## 4. 免责声明与时效（必须）

本简报基于公开信息整理与分析，不构成投资、法律或合规建议。信息可能随政策与市场变化调整，以官方与当日信息为准。
